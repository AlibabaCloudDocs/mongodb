# ModifyDBInstanceNetExpireTime {#doc_api_Dds_ModifyDBInstanceNetExpireTime .reference}

调用ModifyDBInstanceNetExpireTime接口延长MongoDB实例的经典网络保留时长。

调用该接口时实例须满足以下条件：

-   实例状态处于运行中。
-   实例的网络处于混访模式。

**说明：** 本接口适用于副本集实例和分片集群实例，暂不支持单节点实例。
