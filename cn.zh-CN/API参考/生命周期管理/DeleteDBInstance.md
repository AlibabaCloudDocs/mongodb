# DeleteDBInstance

调用DeleteDBInstance接口释放MongoDB实例。

调用该接口时，实例必须满足以下条件。

-   实例状态为运行中；
-   实例付费类型为按量付费。

**说明：** 实例释放后数据将无法找回，请谨慎操作。
