# SwitchDBInstanceHA {#doc_api_Dds_SwitchDBInstanceHA .reference}

调用SwitchDBInstanceHA接口切换实例中的主备节点。

调用本接口时，实例状态要求为运行中。

**说明：** 

 

-   本接口适用于副本集实例和分片集群实例，暂不支持单节点实例。

-   副本集实例以实例单位进行切换，分片集群实例以Shard为单位进行切换。
